# Música de Fundo

## Arquivo Necessário

Coloque um arquivo de música neste diretório com o nome:
- `shopping_background.mp3` ou
- `shopping_background.ogg`

## Formatos Suportados

O pygame suporta os seguintes formatos:
- MP3
- OGG/Vorbis (recomendado para melhor compatibilidade)
- WAV (não recomendado para música longa devido ao tamanho)

## Sugestões de Música

Para uma experiência de compras agradável, considere músicas:
- Instrumentais suaves
- Jazz/Bossa Nova
- Lo-fi
- Música ambiente/relaxante
- Volume recomendado: baixo (não deve distrair)

## Como Adicionar

1. Baixe uma música de fundo (certifique-se de ter os direitos)
2. Renomeie para `shopping_background.mp3` ou `shopping_background.ogg`
3. Coloque neste diretório
4. Reinicie o sistema

A música tocará automaticamente em loop enquanto o usuário navega pelo sistema.
